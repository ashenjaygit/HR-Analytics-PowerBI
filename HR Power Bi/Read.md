🚀 Project Overview

This dashboard helps organizations understand workforce trends such as employee distribution, promotion eligibility, and service years.

The project involves:

Data cleaning and transformation

Data modeling and DAX calculations

Interactive dashboard design

Data visualization for HR insights

🔍 Key Features

Employee overview with total employee count and gender distribution

Promotion eligibility and retrenchment status analysis

Service years distribution visualization

Job level breakdown of employees

Distance from home category insights

Interactive filters to explore data dynamically

⚙️ Tools & Technologies

Microsoft Power BI

Power Query – Data cleaning and transformation

DAX (Data Analysis Expressions) – Calculations and metrics

Data Visualization Techniques
